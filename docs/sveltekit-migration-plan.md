# SvelteKit Migration Status

The application has completed its migration to SvelteKit.

- Pages are implemented in `src/routes`.
- API handlers are SvelteKit `+server.js` routes.
- Shared client, server, data, locale, and style resources live under `src/lib`.
- Vite and `@sveltejs/adapter-vercel` provide development and deployment builds.
- The PWA is generated by `vite-plugin-pwa`.
- Tailwind CSS provides the styling pipeline.

The legacy Next.js/React source tree, configuration, build output, and generated service-worker assets were removed after the SvelteKit routes were verified.
